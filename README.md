# ⚖️ Legalizer | Next-Gen Legal Intelligence

Legalizer is a premium, enterprise-grade Legal Document Management System (LDMS) designed to streamline legal operations for Citizens, Advocates, and Administrators. It combines AI-driven research with high-fidelity document drafting and secure evidence management.

![Legalizer Banner](https://images.unsplash.com/photo-1589829545856-d10d557cf95f?auto=format&fit=crop&q=80&w=1200)

## 🌟 Core Features

### 🚀 Smart Drafting Suite
Our document editor isn't just a text box. It's a **Smart Blueprint Engine** that automatically scans drafts for `[PLACEHOLDERS]` and provides a custom Wizard to populate document fields in seconds. 
- Real-time legal syntax verification.
- One-click export to PDF.
- Integrated with our Marketplace of verified legal templates.

### 🧠 Legal AI Assistant
Consult our specialized AI knowledge engine trained on the **Indian Constitution**. 
- Instant references to specific Articles and Fundamental Rights.
- Intelligent discovery of legal precedents.
- Context-aware drafting suggestions.

### 📊 Executive Dashboard
A multi-role nerve center for managing legal operations:
- **Clients**: Connect with verified advocates and track case statuses.
- **Advocates**: Manage representation requests and document lifecycles.
- **Admins**: Verify advocate credentials and monitor system-wide compliance scores.

### 🔐 Evidence Vault & Signatures
- **High-Security Vault**: Encrypted storage for critical case files and binary evidence.
- **Digital signature Pad**: A high-resolution canvas tool to generate transparent PNG signatures ready for court-ready documents.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org) (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS with a bespoke "Glassmorphism" Design System
- **Charts**: [Chart.js](https://www.chartjs.org/) for real-time utilization metrics
- **State Management**: React Context API (Simulation Engine)
- **Storage**: Local-first simulation for sensitive data persistence

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Access the App**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `/src/app`: Next.js App Router pages and layouts.
- `/src/components`: Reusable UI components (Charts, Navigation, etc.).
- `/src/context`: Simulation Engine state management.
- `/src/data`: Knowledge base (Constitution, Templates, Advocates).
- `/src/styles`: Global design tokens and animations.
